# cmz-tweet

Example of styling a tweet with [`cmz`](https://github.com/joshwnj/cmz)

Based on <https://github.com/geelen/css-components-demo>

## Try it out

- clone the repo
- `npm install`
- `npm run build`
- `open dist/index.html`

You can also run `npm run hot` for hot-reloading.
